# Geinel's Portfolio

A personal portfolio with a Windows 97 desktop aesthetic. Built to showcase my projects, experience, skills, and certifications through a retro UI complete with draggable windows, a taskbar, physics simulations, and an animated preloader.

## Stack

- **Next.js 16** — App Router, static export
- **React 19** — UI
- **TypeScript** — type safety throughout
- **Tailwind CSS v4** — styling
- **Framer Motion** — animations and transitions
- **Matter.js** — physics engine for interactive elements

## Run locally

```bash
npm install
npm run dev
```
